    Smart Glove Translator

    This Android app translates sign language into text and speech using a smart glove.
    Developed with Java and Android Studio, it connects via Bluetooth to an ESP32 microcontroller that sends real-time sensor data.
    A machine learning model classifies static hand gestures to recognize signs accurately.
    The app converts recognized gestures into speech with a text-to-speech feature, enhancing accessibility for users.
    It includes a multi-screen user interface for translation, gesture previews, and customizable settings,
    allowing users to manage and personalize gesture mappings for a better user experience and greater flexibility.

    What I did

    • Developed Bluetooth communication to receive real-time sensor data from the smart glove.
    • Integrated a machine learning model to accurately classify static hand gestures.
    • Created a multi-screen UI with translation, gesture preview, and customizable settings for better user control.
